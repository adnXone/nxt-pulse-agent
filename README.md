# nxt-pulse-agent v0.2.0

A highly proactive OpenClaw skill (NXT) that manages user energy levels and task prioritization with ADHD micro-stepping support.

## Features
- **Visual Energy Pulse**: 🟢/🟡/🔴 indicators.
- **Just 2 Minutes Mode**: Break through ADHD paralysis.
- **Audit Trail**: Full transparency for admin reviews.
- **Logic Engine**: Local state management via `pulse.js`.

## Installation
```bash
clawhub install adnxone/nxt-pulse-agent
```
